# Tips & Best Practices

1. Use relevant image
2. Add visual hierarchy
3. Limit content
4. Avoid embedded links
5. Differentiate actions
6. Resist lines for grouping

# Aesthetic & Modern UI cards

1. Use rounded corners
2. Add a light border or soft shadow
